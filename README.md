# spring-thymeleaf-scaffolding
Scaffolding Application Using SpringBoot and Thymeleaf.
Reusable Component

# Technologies !
WEB :
  - Spring Web
  - Spring Security
  - Thymeleaf
  - Bootstrap
  - Jquery

API :
  - Spring Web
  - JPA
  - MySql
